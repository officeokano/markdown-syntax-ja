# markdown-syntax-ja

Markdownの書式まとめ日本語版です。 [markdown-syntax-ja][1] 自体がマークダウン記法で書かれています。

この作品は [MIT ライセンス][2]で提供されています。 [markdown-syntax-ja][1] ファイルにライセンスの表記がありますから、そのままコピーして利用できます。
