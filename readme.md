# markdown-syntax-ja

Markdownの書式まとめ日本語版です。 [markdown-syntax-ja][1] 自体がマークダウン書式で書かれています。

この作品は、クリエイティブ・コモンズの [表示 - 継承 4.0 国際 ライセンス][2]で提供されています。 [markdown-syntax-ja][1] ファイルにライセンスの表記がありますから、そのままコピーして利用できます。
